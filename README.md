# floatboat

Floatboat is a tiny Ruby library for peeking inside Ruby's IEEE 754 floating point representation.

## Installation

Add this line to your application's Gemfile:

    gem 'floatboat'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install floatboat
